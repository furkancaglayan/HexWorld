Version 1.2

Changes:
-Drawing is optimized.
-Removed HexWorld Logger. Instead, errors show in dialog popups.
-UI is renewed.
-Map size is now chosen with keywords like small, large etc. instead of integers.
-Hexagonal map type is removed.

Features:
-Model Viewer: Easily inspect your prefabs with a click in the editor.
-Post Processing Settings: Game ready lightning and post processing effects.Added by one button.
-HexWorld Effect: Effect asset for HexWorld. Create it and add to your scene as custom profiles.
[Currently works with gradient sky]








Version 1.1

Fixes:
-Removed a deprecated function from Unity version 2018.3 and newer.

-Fixed a bug about applying random rotation when using 'Fill all' and 'Fill Rest' options.

-Now you can save your map data as HexWorldStaticData and continue
painting your map whenever you want. Instead of old save option, there is now 'Export as Prefab' option
